#1099 문제


P1099 : [기초-종합+배열] 성실한 개미


```python
# 내 풀이

## input값 넣기
mat = []
for i in range(10) :
    mat.append(list(map(int, input().split())))

## (1,1)에 9를 넣고, 오른쪽이 0이라면 y축으로 +1, 오른쪽이 1이라면 x축으로 +1, 
## 만약에 내 위치가 (8,8)이거나 또는 값이 2라면 그 자리를 9로 치환하고 break
x = 1
y = 1
while True :
    if mat[x][y+1] == 0 :
        mat[x][y] = 9
        y = y+1
    elif mat[x][y+1] == 1 :
        mat[x][y] = 9
        x = x+1
    if x == 8 and y == 8 :
        mat[x][y] = 9
        break
    if mat[x][y] == 2:
        mat[x][y] = 9
        break
        
## 리스트를 output으로
for i in range(10) :
    for j in range(10) :
        print(mat[i][j], end = ' ')
    print('')
```
