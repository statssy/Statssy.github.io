## [백준 문제] P2609-최대공약수와 최소공배수


```python
# 최대공약수 / 최소공배수
def gcd(a, b):
    if b == 0 :
        return a;
    else :
        return int(gcd(b, a%b))

def lcm(a, b, g) :
    l = g*(a/g)*(b/g)
    return int(l)

a, b = map(int, input().split())

g = gcd(a, b)
l = lcm(a, b, g)
print(g)
print(l)
```

    10 12
    2
    60


## [백준문제] P1934-최소공배수


```python
def gcd(a, b):
    if b == 0 :
        return a;
    else :
        return int(gcd(b, a%b))

def lcm(a, b, g) :
    l = g*(a/g)*(b/g)
    return int(l)

t = int(input())

for i in range(t) :
    a, b = map(int, input().split())
    g = gcd(a, b);
    l = lcm(a, b, g);
    print(l)


```

    3
    1 10000
    10000
    2 14
    14
    3 15
    15
