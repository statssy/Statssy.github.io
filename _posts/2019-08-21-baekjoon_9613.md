## [백준 문제]P9613-GCD합


```python
def gcd(a, b):
    if b == 0 :
        return a;
    else :
        return gcd(b, a%b)

n = int(input())
for i in range(n) :
    arr = list(map(int, input().split()))
    sum = 0
    for j in range(1, arr[0]+1) :
        for k in range(1, arr[0]+1) :
            if j != k :
                sum = sum + gcd(arr[j], arr[k])
    print(int(sum/2))
```
