P1064 : [기초-삼항연산] 정수 세 개 입력받아 가장 작은 수 출력하기


```python
# 내 답안 1
a, b, c = map(int, input().split())
x = a, b, c
y=[]
for i in x :
    y.append(i)
print(min(y))
```

    1 2 3
    




    1




```python
# 내 답안 2
a, b, c = map(int, input().split())
x = a, b, c
y=[]
for i in x :
    y.append(i);

minimum = y[0]

for num in y[1:] :
    if num < minimum :
        minimum = num
print(minimum)
```

    1 2 3
    1

P1068 : [기초-조건/선택실행구조] 정수 한 개 입력받아 평가 출력하기  
90 ~ 100 : A  
70 ~ 89 : B  
40 ~ 69 : C  
0 ~ 39 : D  


```python
a = int(input())
if 90 <= a <= 100 :
    print('A')
elif 70 <= a <= 89 :
    print('B')
elif 40 <= a <= 69 :
    print('C')
elif 0 <= a <= 39 :
    print('D')
else :
    print('N')
```

    20
    D
